How it works

  1. Match job title, experience, and keywords against comparable postings.
  2. Score each posting from 0 to 100 using simple weighted heuristics.
  3. Use the strongest salary-visible postings to estimate a range and remove obvious outliers.
  4. Show recruiters the evidence so they can make a fast, explainable call.
